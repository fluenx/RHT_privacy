Privacy Policy – Resume Handling Tool

1. Overview

The Resume Handling Tool is a private internal Chrome extension developed exclusively for the HR team at Logicose. The extension is designed to streamline recruitment workflows by automating the extraction of candidate information from resumes and populating relevant fields on the Logicose RMS (Recruitment Management System) software. It is not intended for public use.

2. Data Access and Use

The extension accesses and processes the following data to provide its core functionality:

  • Personally Identifiable Information (PII): Extracts candidate names, email addresses, and phone numbers from uploaded resume files (PDF, DOCX, and Images).
  • Website Content: Accesses Logicose's internal RMS pages to automatically fill in the extracted candidate details and check for existing candidate profiles.

This data is processed locally within the user’s browser session using the offscreen API and is used strictly for internal HR recruitment operations.

3. Data Storage

The extension utilizes Chrome’s local storage API to store:

  • Temporary workflow state.
  • User-specific console logs for debugging and verification purposes.
  • Metadata for the manual selection interface.

No resume content or personal data is transmitted to or stored on external servers. All parsing and extraction are performed on-device.

4. Data Sharing

The extension does not sell, rent, or share user data with any third parties. It does not use external analytics or tracking scripts. Data remains localized to the user's browser and the Logicose internal RMS portal.

5. Security

The extension operates within Chrome’s strict security model and requests only the minimal permissions required for functionality (e.g., offscreen for parsing, activeTab for form filling). No remote code is executed; all processing libraries are bundled locally within the extension package.

6. Restricted Data Types

The extension does not collect or access passwords, payment information, health data, browsing history, or location data.

7. Changes to This Policy

This privacy policy may be updated to reflect changes in functionality or internal compliance requirements. Updates will apply only to internal users at Logicose.
